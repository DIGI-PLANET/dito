## 🚀 **Quick Start**

### **For Developers**
```bash
# Install DITO SDK
npm install dito-sdk

# Start building
import { DitoClient } from 'dito-sdk';
const dito = new DitoClient('https://gateway.dito.guru', 'your-token');
```

### **For AI Agents**
```javascript
import { DitoAgent } from 'dito-sdk';
const agent = new DitoAgent(dito);

// Natural language interactions
const status = await agent.check2FAStatus();
const talents = await agent.suggestTalents();
```
